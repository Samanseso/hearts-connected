BEYOND THE SCREEN
FULL PROJECT DOCUMENTATION

1. PROJECT SUMMARY

Beyond the Screen is a thesis-inspired interactive visual novel about modern relationships in the age of social media. The game presents branching narrative scenarios where the player makes dialogue choices that affect emotional outcomes, relationship health, ending type, and route score.

The project was built as a playable narrative experience rather than a static presentation. It combines story routes, decision-based outcomes, stat tracking, route grading, unlockable content, voice-line support, and a dashboard-style archive system.


2. PROJECT GOALS

- Present relationship issues in an interactive format.
- Translate research themes into playable case studies.
- Encourage reflection through branching endings and route lessons.
- Make the experience feel like a game intro and visual novel, not a dashboard.
- Track progress, endings, and route performance for replay value.


3. CORE GAME CONCEPT

The player enters a story hub and selects one of several relationship scenarios. Each scenario focuses on a different issue connected to social media, communication, or emotional expectations. The player makes choices during the route, which influence:

- Trust
- Communication
- Contentment
- Anxiety
- Pressure
- Self-Respect
- Commitment

At the end of each route, the player receives:

- an ending title
- an ending message
- a takeaway or lesson
- a quote
- a route score out of 100
- a route grade


4. SCENARIO LIST

The game currently contains 7 main playable scenarios:

1. Social Media Influence on Relationship Expectations
2. Online Jealousy and Trust Issues
3. Peer Pressure in Dating
4. Comparing Relationships Online
5. Commitment Decisions
6. Dating Norms and Mixed Signals
7. Digital Availability and Emotional Support

Each scenario currently has 3 endings, for a total of 21 endings across the full game.

11. CHARACTER SHOWCASE SYSTEM

The character showcase acts as an unlockable archive.

Rules:
- If the player has completed the related route, the character pair is unlocked.
- If the player has not completed the route, the entry stays marked Locked.

Unlocked content includes:
- character profile
- sprite model viewer
- reaction face switching
- generated voice line list

This system helps turn story completion into visible progression and gives each route additional collectible value.

17. HOW AUDIO GENERATION WORKS

For line-by-line voice generation:
1. Prepare docs/voice-prompts.csv
2. Run the line generation script
3. Generated files are placed under public/asset/audio/lines/<scene-id>/
4. Generate or refresh manifest.json
5. The in-game player uses the manifest to match each spoken line

Optional notes:
- MP3 generation may require ffmpeg
- Gemini TTS requires an API key in .env
- A Puter browser-based generator tool also exists for browser-side generation


18. CURRENT DESIGN DECISIONS

21. CONCLUSION

Beyond the Screen is a modern interactive visual novel built with Next.js, React, TypeScript, Tailwind CSS, and narraleaf-react. It combines branching storytelling, research-inspired themes, route grading, voice integration, unlockable character content, and polished presentation into a playable educational narrative prototype.

It is both a game and a reflective storytelling tool, designed to help players explore how social media affects expectations, emotional communication, and relationship decisions.
